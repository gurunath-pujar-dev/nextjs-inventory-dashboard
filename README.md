---

## ⚙️ Execution steps:

Clone & Install dependencies:

```bash
git clone https://github.com/gurunath-pujar-dev/nextjs-inventory-dashboard.git
cd nextjs-inventory-dashboard
npm install
```


Run command:
```bash
npm run dev
```
